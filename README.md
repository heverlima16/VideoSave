# VideoSave
Guarda Videos docker

## About
A professional video downloader supporting multiple platforms (YouTube, TikTok, Instagram, etc.) with a modern dark-themed UI.

## Setup
1. Install dependencies: `npm install`
2. Ensure `yt-dlp` is installed on your system
3. Run the server: `node server.js`
4. Access the app at `http://localhost:3000`
